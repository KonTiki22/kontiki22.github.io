# 3D Галерея экспонатов выставки «Коренные народы Сибири»

Интерактивная 3D галерея для просмотра фигурок семьи Фридман из собрания ГБУК «ИОДНТ».

## Особенности

- Просмотр 3D моделей фигурок с возможностью вращения (мышь и касания)
- Зум колесиком мыши или жестом щипка на мобильных устройствах
- Красивый градиентный фон в оттенках серого
- Меню для переключения между 18 экспонатами
- Адаптивный интерфейс для десктопов и мобильных устройств

## Использование

1. Откройте файл `index.html` в современном браузере
2. Используйте мышь для вращения моделей:
   - Перетаскивание влево/вправо - вращение по горизонтали
   - Перетаскивание вверх/вниз - вращение по вертикали (инвертировано)
   - Колесико мыши - приближение/удаление
3. На мобильных устройствах:
   - Свайп одним пальцем - вращение
   - Щипок двумя пальцами - зум
